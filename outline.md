## Visual & Technical Recommendations

**Design system:**
- Clean, professional: Think Stripe Docs or HashiCorp Learn pages
- Primary blue + healthcare green palette
- Heavy use of diagrams, tables, downloadable assets
- Dark mode toggle
- Excellent mobile UX (progressive web app)

**Key success metric for v1:**
Can someone download a compliance pattern template and implement a basic version in 1 hour? If yes, the site works.
